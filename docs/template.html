Вставка переменных

Переменные в шаблоне вставляются с помощью двойных фигурных скобок в нужное место шаблона, например:

Привет, {{ member.name }}!

В примере выше переменная member.name вставлена в нужное место шаблона и обозначена двойными фигурными скобками. Это значит, что будет заменено на ник пользователя, то есть в чат распечатается следующее:

Привет, Пользователь!
